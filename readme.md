
# StoryNote

简易博客

后端由 Python 下的 tornado 框架支撑。

前端使用 vue 全家桶 webpack 方案。

数据库使用 SQLite，备份时可以拷了单文件就走。

**只在 Python3.5+ 环境下开发与使用过，基本能保证3.3+的正常运作**
